### MATAMOROS QR

<p aling="justify">
  This project is an application designed to register assets by area during QR scanning or manual input, store them temporarily in JSON, and later save them into a local database
</p>

### TECHNOLOGIES USED
- Python.
    - NiceGUI.
- SQLite.
- JSON.

### FEATURES
- Asset capture by area.
- QR-style fast input support.
- Local JSON buffer to prevent data los.
- SQLite database with relational structure.
- Manual synchronization to database.
- Duplicate detection.
- Real-time asset visualization by area.
